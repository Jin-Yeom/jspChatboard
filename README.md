# jspChatboard
실시간 채팅기능과 게시판기능을 혼합해놓은 jsp웹사이트입니다.
실시간채팅과 메시지 그리고 프로필사진을 추가할 수 있습니다.
게시판 또한 이미지파일을 첨부할 수 있으며 다운로드 할 수 있게 만들었습니다. 
이 기능들은 회원가입 및 로그인을 하지 않으면 사용할 수 없습니다.

# 사용 기술
<li><a>Java, Html, xml, Servlet, Ajax, Json, bootstrap</a></li>
